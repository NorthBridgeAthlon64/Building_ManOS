# 答辩 PPT（PPT组）

## 建议结构（8~10 页）

1. 封面（项目名、团队、课程）
2. 背景与目标
3. 功能模块总览
4. 系统架构 / 类图
5. 核心流程（查询、购买、折扣）
6. 终端演示（截图或录屏）
7. 测试与成果
8. 团队分工与总结
9. Q&A

## 子目录

| 目录 | 用途 |
|------|------|
| `slides/` | `.pptx` 终稿与迭代版本 |
| `assets/` | 配图、Logo、演示截图 |
| `scripts/` | 演讲稿、时间分配、分工台词 |

## 协作注意

- 术语与 `docs/`、`项目计划.md` 保持一致
- 演示页占比建议 ≥ 40%
- 终稿命名：`Building_ManOS-答辩-v1.pptx`
