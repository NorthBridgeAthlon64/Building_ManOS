# 答辩 PPT（PPT 组）

## 核心文档 ★

| 文档 | 说明 |
|------|------|
| [**答辩 PPT 大纲**](./答辩PPT大纲.md) | **6 页口述** + 可选 Q&A，总时长 **≤ 5 分钟** |
| [**演讲稿**](./scripts/演讲稿.md) | 5 分钟台词（演示 2 分钟）、Q&A 附录 |

## 建议结构（≤ 5 分钟）

1. 封面（15 秒）
2. 项目概述（背景 + 功能 + 课程要求，合并一页）
3. 技术设计（架构 + 数据库 + 购买流程，合并一页）
4. **现场演示**（2 分钟，3 步）★
5. 团队分工（25 秒）
6. 总结与致谢（15 秒）
7. Q&A 备注页（不口述，可选）

## 子目录

| 目录 | 用途 |
|------|------|
| `slides/` | `.pptx` 终稿与迭代版本 |
| `assets/` | 配图、Logo、演示截图 |
| `scripts/` | 演讲稿、时间分配、分工台词 |

## 协作注意

- 术语与 `docs/`、`项目计划.md` 保持一致
- 演示页占比建议 ≥ 40%（5 分钟版中演示固定 2 分钟）
- 终稿命名：`Building_ManOS-答辩-v1.pptx`
- 架构图、E-R 图可从 `docs/design/` 导出至 `assets/`
