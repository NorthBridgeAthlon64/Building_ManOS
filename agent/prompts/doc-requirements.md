# 文档撰写 Prompt 模板

供文档组在 Cursor 中使用，@ 本文件后发送。

---

## 需求分析章节

```
@项目计划.md
@docs/requirements/课程要求对照.md

请撰写「需求分析」章节（课程设计报告用）：
- 项目背景与目标
- 功能需求列表（楼盘/房屋/查询/购买）
- 非功能需求（控制台、MySQL、分层架构）
- 体现团队分工占位：[成员姓名 - 负责内容]

语气：课程报告正式文体，中文。
```

---

## 系统设计章节

```
@项目计划.md
@sql/schema.sql

请撰写「系统设计」：
- 四层架构说明（cli/service/dao/数据库）
- 主要类职责表
- E-R 图文字描述（building/house/sale_record 关系）
- 购买流程步骤

附 Mermaid 类图或流程图（若合适）。
```

---

## 注释合规审查

```
@docs/design/代码注释规范.md

审查以下文件的 JavaDoc 是否符合团队规范，列出缺失项：
@<文件路径>
```
