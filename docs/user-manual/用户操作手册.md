## 1. 系统简介

本系统为**命令行控制台**程序，用于管理房地产楼盘、房屋信息，支持查询在售房源、应用折扣完成购买，并将数据保存至 MySQL 数据库。

## 3. 安装与配置

### 3.1 获取代码

```powershell
git clone https://github.com/NorthBridgeAthlon64/Building_ManOS.git
cd Building_ManOS
```

### 3.2 创建数据库

在 MySQL 中执行：

```sql
SOURCE d:/Java_Project/Building_ManOS/sql/schema.sql;
SOURCE d:/Java_Project/Building_ManOS/sql/init-data.sql;
```

> 路径请按本机实际位置修改。  
> **注意**：当前仓库中 `init-data.sql` 演示 INSERT 仍为注释，需技术组补充后再导入演示数据。

### 3.3 配置数据库连接

编辑 `src/main/resources/database.properties`（与课堂 `database.properties` 同模式）：

```properties
driver=com.mysql.cj.jdbc.Driver
url=jdbc:mysql://localhost:3306/building_manos?useUnicode=true&characterEncoding=UTF-8&serverTimezone=Asia/Shanghai&useSSL=false
user=你的用户名
password=你的密码
```

> 也可通过环境变量 `DB_URL`、`DB_USER`、`DB_PASSWORD` 覆盖（见 `Java技术框架.md` §4.1）。

### 3.4 编译项目

```powershell
mvn compile
```

## 6. 功能操作指南

### 6.1 楼盘管理

**新增楼盘**：输入楼盘名称、占地面积、地址、开发商（可选）、备注（可选）。

**修改楼盘**：先选择楼盘编号，再修改字段。

**删除楼盘**：若该楼盘下仍有房屋，系统会拒绝删除。

**查看列表**：显示所有楼盘的编号、名称、地址等。

### 6.2 房屋管理

**新增房屋**：

1. 选择所属楼盘  
2. 输入楼号、房号、面积、单价  
3. 系统自动计算总价（面积 × 单价）  
4. 新房屋默认状态为「在售」

**注意**：同一楼盘下楼号+房号不能重复。

**修改/删除**：仅「在售」房屋允许修改或删除。

### 6.3 房屋查询

支持组合或单项条件（以实际菜单为准）：

- 楼盘名称（模糊）  
- 楼号  
- 价格区间  
- 面积区间  
- 销售状态（在售 / 已售）

查询结果以表格形式在终端显示。

### 6.4 房屋购买

1. 系统列出或在售房屋中选择一套  
2. 查看房屋详情与原价  
3. 选择折扣类型：**档位比例折扣** 或 **档位满减**（系统按原价自动匹配档位）  
4. 输入客户姓名  
5. 确认成交信息（展示原价、档位参数、实付金额）  
6. 成交后房屋变为「已售」，并生成销售记录  

**已售房屋无法再次购买。** 一套房最多一条成交记录（数据库唯一约束）。

### 6.5 销售记录

查看历史成交：房屋信息、原价、折扣、实付金额、客户、成交时间。

## 8. 常见问题

### Q1：启动报错「Communications link failure」

- 检查 MySQL 服务是否启动  
- 检查 `database.properties` 中 URL、用户名、密码  
- 确认数据库 `building_manos` 已创建  

### Q2：中文乱码

- 终端编码设为 UTF-8  
- 确认数据库与连接串使用 `utf8` / `utf8mb4`  

### Q3：Maven 找不到命令

- 安装 Maven 并配置 `PATH`  
- 或使用 IDE 运行 `Main` 类  

### Q4：菜单输入无效

- 仅输入数字选项，按回车确认
