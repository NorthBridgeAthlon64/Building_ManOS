# 项目文档（文档组）

本目录存放课程大作业全部书面文档，按阶段分子目录维护。

## 子目录说明

| 目录 | 内容 | 建议负责人 |
|------|------|------------|
| [requirements/](./requirements/) | 需求分析、用例图、功能列表、数据字典 | 需求负责人 |
| [design/](./design/) | 概要/详细设计、类图、流程图、包图 | 设计负责人 |
| [test/](./test/) | 测试用例、测试报告、截图 | 测试负责人 |
| [user-manual/](./user-manual/) | 用户操作手册（配合答辩演示） | 文档整合 |
| [report/](./report/) | 课程设计报告终稿 | 报告主编 |

## 命名规范

- 文件名：`模块名-文档类型-v版本.md` 或按学校模板命名
- 图片：放在对应子目录的 `images/` 下
- 图表源文件（`.drawio` / `.uml`）与导出 PNG 一并提交
