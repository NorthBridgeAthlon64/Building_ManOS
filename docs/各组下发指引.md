## 二、技术组 — 必读与任务

### 必读文档（按顺序）

1. [Java技术框架.md](./design/Java技术框架.md) — **最高优先级**，包结构、接口签名、开发顺序  
2. [数据库设计.md](./design/数据库设计.md) — 表结构、DAO-SQL  
3. [代码注释规范.md](./design/代码注释规范.md) — 每类 JavaDoc + `@author`  
4. [课堂知识点对照.md](./design/课堂知识点对照.md) — 参考课堂示例写法  
5. [sql/schema.sql](../sql/schema.sql)、[sql/init-data.sql](../sql/init-data.sql)

### 本周任务（第 1 周）

| 优先级 | 任务 | 验收标准 |
|--------|------|----------|
| P0 | 实现 `DBConfig` + 连通测试 | `mvn compile` 通过，能连 MySQL |
| P0 | `model` + `BuildingDao` + `BuildingService` | 楼盘增删改查跑通 |
| P0 | `MenuController` 主菜单骨架 | 能显示 5 项菜单 |
| P1 | `HouseDao` + `HouseService` | 房屋 CRUD + 总价计算 |

### 硬性规定（违反即返工）

- 包名：`com.building.manos`
- 分层：cli → service → dao，**禁止 cli 直调 dao**
- 配置：`src/main/resources/database.properties`（键名 `driver/url/user/password`）
- 类名：`DBConfig`（不是 DbConfig）
- 禁止 GUI / Web / `repository` 包

### 参考代码

`docs/course_example_java_learn/` — 鲜花商店分层、JDBC 示例（已 UTF-8）
