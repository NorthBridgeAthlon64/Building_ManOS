### 本周任务

| 优先级 | 任务 | 验收标准 |
|--------|------|----------|
| P0 | 实现 `DBConfig` + 连通测试 | `mvn compile` 通过，能连 MySQL |
| P0 | `model` + `BuildingDao` + `BuildingService` | 楼盘增删改查跑通 |
| P0 | `MenuController` 主菜单骨架 | 能显示 5 项菜单 |
| P1 | `HouseDao` + `HouseService` | 房屋 CRUD + 总价计算 |

### 硬性规定

- 包名：`com.building.manos`
- 分层：cli → service → dao，**禁止 cli 直调 dao**
- 配置：`src/main/resources/database.properties`
- 禁止 GUI / Web

## 统一约定（勿各写各的）

| 项 | 统一规定 |
|----|----------|
| 数据库名 | `building_manos` |
| 表名 | `building`、`house`、`sale_record` |
| 配置文件 | `database.properties`（键名：driver/url/user/password） |
| 连接类 | `config/DBConfig.java` |
| 房屋状态 | `ON_SALE` / `SOLD` |
| 答辩时长 | ≤ 5 分钟 |
| 演示步骤 | 3 步：查在售 → 购买九折 → 验证已售 |
