## 1. building（楼盘表）

| 字段名 | 类型 | 空 | 说明 |
|--------|------|----|------|
| id | VARCHAR(32) | N | 主键，楼盘编号 |
| name | VARCHAR(100) | N | 楼盘名称 |
| land_area | DECIMAL(12,2) | N | 占地面积，单位：㎡ |
| address | VARCHAR(200) | N | 地址 |
| developer | VARCHAR(100) | Y | 开发商 |
| remark | VARCHAR(500) | Y | 备注 |
| created_at | DATETIME | Y | 创建时间，默认当前时间 |

---

## 2. house（房屋表）

| 字段名 | 类型 | 空 | 说明 |
|--------|------|----|------|
| id | VARCHAR(32) | N | 主键，房屋编号 |
| building_id | VARCHAR(32) | N | 外键，所属楼盘 id |
| building_no | VARCHAR(20) | N | 楼号，如「3栋」 |
| room_no | VARCHAR(20) | N | 房号，如「1201」 |
| area | DECIMAL(10,2) | N | 面积，单位：㎡ |
| unit_price | DECIMAL(12,2) | N | 单价，单位：元/㎡ |
| total_price | DECIMAL(14,2) | N | 总价，元 |
| status | VARCHAR(20) | N | ON_SALE / SOLD |
| sold_at | DATETIME | Y | 售出时间 |

**唯一约束**：`(building_id, building_no, room_no)`

---

## 3. sale_record（成交记录表）

| 字段名 | 类型 | 空 | 说明 |
|--------|------|----|------|
| id | VARCHAR(32) | N | 主键 |
| house_id | VARCHAR(32) | N | 外键，房屋 id |
| original_price | DECIMAL(14,2) | N | 成交原价，元 |
| discount_type | VARCHAR(50) | Y | 折扣类型，如 PERCENTAGE |
| discount_value | DECIMAL(10,4) | Y | 折扣参数 |
| final_price | DECIMAL(14,2) | N | 实付金额，元 |
| customer_name | VARCHAR(50) | Y | 客户姓名 |
| sold_at | DATETIME | Y | 成交时间 |

**唯一约束**：`house_id`，一套房屋最多对应一条成交记录。

## 4. 状态码

| 代码 | 含义 |
|------|------|
| ON_SALE | 在售 |
| SOLD | 已售 |

## 5. 折扣类型

按房屋**原价档位**计算优惠（实现见 `discount/PriceTier.java`）。

| discount_type | 含义 | discount_value 含义 | 档位示例 |
|---------------|------|---------------------|----------|
| PERCENTAGE | 档位比例折扣 | 该次成交使用的折扣率 | 150 万房源为 0.97 |
| THRESHOLD | 档位满减 | 该次成交减免的金额（元） | 150 万房源为 50000 |

**档位一览**：

| 原价（元） | 比例折扣 | 满减 |
|------------|----------|------|
| &lt; 100 万 | 不打折 | 减 2 万 |
| 100 万 ~ 300 万 | 97 折 | 减 5 万 |
| ≥ 300 万 | 92 折 | 减 15 万 |
