# JAVA 高级编程大作业 — 课程要求对照表

> 本文档将学校官方要求与本项目一一对应，供文档组写入报告、PPT 组制作答辩材料时引用。

## 官方要求原文要点

| 序号 | 要求 | 本项目落实方式 |
|------|------|----------------|
| 1 | 使用 **Java** 完成 | Java 17 + Maven，仅 `src/` 下 Java 源码 |
| 2 | **结合数据库**进行数据存取（不限数据库） | MySQL 8.x + JDBC（`sql/` 建表脚本，`dao` 层访问） |
| 3 | **分层架构**设计 | 表示层(cli) → 业务层(service) → 数据访问层(dao) → 数据库 |
| 4 | 只能是 **控制台程序** | `cli` 包菜单驱动，禁止 Swing/JavaFX/Web/安卓等 |
| 5 | 制作 **PPT** 并 **撰写文档**（按模板） | `ppt/` + `docs/report/`（含团队分工章节） |
| 6 | **答辩演示** | 终端现场演示 + 演讲稿（`ppt/scripts/`） |
| 7 | 代码必须加 **注释**，每个类含编写者、完成时间、功能 | 见 `docs/design/代码注释规范.md` |
| 8 | 详细分析与设计，系统尽量 **符合实情** | 调研房地产销售流程，扩展客户/销售员/成交记录等 |
| 9 | 两组代码完全一样则 **均为零分** | 见 `项目计划.md` 第十章「原创性要求」 |

## 禁止项（违反即不符合要求）

- Windows 窗体程序（Swing、JavaFX、AWT 图形界面）
- Java Web（Servlet、Spring Boot 提供 HTTP 接口等）
- Android / 微信小程序 / 其他非控制台形式

## 提交物清单（按课程模板调整）

- [ ] 源代码（含完整类注释）
- [ ] 数据库脚本（`sql/schema.sql`、`sql/init-data.sql`）
- [ ] 课程设计文档（含团队分工）
- [ ] 答辩 PPT
- [ ] 运行说明（`README.md`）
