## 官方要求原文要点（及本组落实）

| 序号 | 要求 | 本项目落实方式 |
|------|------|----------------|
| 1 | 使用 **Java** 完成 | Java 17 + Maven |
| 2 | **结合数据库**进行数据存取 | MySQL + JDBC（`sql/` + `dao`） |
| 3 | **分层架构**设计 | cli/api → service → dao → MySQL |
| 4 | **控制台程序**（课程基线） | `cli` 菜单驱动，**答辩兜底必须可运行** |
| 5 | **撰写文档**（按模板） | **[大作业报告.md](../report/大作业报告.md)**（含分工）；PPT **已取消** |
| 6 | **答辩演示** | 终端或 Web 现场演示（见用户手册 §7） |
| 7 | 代码必须加 **注释** | 见 `代码注释规范.md`；`@author` 对齐分工表 |
| 8 | 详细分析与设计，尽量符合实情 | 楼盘/房屋/成交、档位折扣、事务与唯一约束 |
| 9 | 两组代码完全一样则 **均为零分** | 见 `项目计划.md` 原创性要求 |

## 扩展说明（相对最初「仅控制台」表述）

本仓库在满足 **cli 可演示** 的前提下，额外实现了 **Javalin API + Vue 前端**，与 cli **共用 service/dao**，便于联调与界面截图。若验收严格按「仅控制台」，**直接演示 cli 即可**，Web 作为加分展示。
